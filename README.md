# procurement-app-version
It points to the latest version of the app

Once the version is updated, the app will automatically reflect the changes.
This ensures that users always have access to the most recent features and bug fixes.
This is particularly useful for organizations that use the app for procurement purposes.

Need to update a field APP_VERSION here:
https://github.com/Viktor-Vrublevski/agent-app/blob/master/app-module/src/main/java/com/procurement/agent/RemoteVersionUploader.java#L26

To update the version, follow these steps:
1. Navigate to the RemoteVersionUploader.java file.
2. Locate the APP_VERSION constant.
3. Update the version number.
4. Commit and push the changes to the repository 'agent-app'.
